# `CustomSelect`

> 含全选功能的 antd select 组件；

## 用法

```tsx
import { CustomSelect } from '@hbwow/components';

<CustomSelect options={[]} allField={null} />;
```

## API

| 参数     | 说明              | 类型 | 默认值 | 版本 |
| -------- | ----------------- | ---- | ------ | ---- |
| allField | 全部选项的value值 | any  | -      |
